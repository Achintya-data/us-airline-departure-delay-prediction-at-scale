# Reports Guide

This folder is organized to separate the main deliverables from the notebook export artifacts.

## Main Files

- `final-report.pdf`: compressed final course report
- `final-pyspark-analysis.html`: merged PySpark analysis export
- `final-pyspark-analysis.md`: markdown export of the merged analysis
- `final-pyspark-analysis.tex`: LaTeX export of the merged analysis

## Notebook HTML Exports

`notebook-html/` contains the four HTML exports for the project notebooks:

- `data-eda-report.html`
- `dataset-etl-report.html`
- `project-pyspark-report.html`
- `scaling-analysis-report.html`

These are preserved for review, but they are marked as generated in `.gitattributes` so GitHub language statistics are not dominated by HTML.

## Notebook Report Sources

`notebook-ipynb/` contains the notebook versions of those four report exports:

- `data-eda-report.ipynb`
- `dataset-etl-report.ipynb`
- `project-pyspark-report.ipynb`
- `scaling-analysis-report.ipynb`
